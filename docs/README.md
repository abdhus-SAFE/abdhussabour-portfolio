# GitHub Pages Setup

This directory contains the GitHub Pages website for the portfolio.

## Files

- `index.html` - Main website landing page
- `styles.css` - Professional styling
- `script.js` - Interactive features

## Deployment Instructions

### Step 1: Enable GitHub Pages
1. Go to repository Settings
2. Navigate to Pages section
3. Select Source: `main` branch, `/docs` folder
4. Click Save

### Step 2: Setup Custom Domain
1. In GitHub Pages settings, add custom domain: `sabouraxiom.ae`
2. Add DNS records to your domain registrar:
   - CNAME: `abdhus-SAFE.github.io` → `sabouraxiom.ae`
3. Wait for DNS propagation (24-48 hours)

### Step 3: Enable HTTPS
1. Check "Enforce HTTPS" in GitHub Pages settings
2. Wait for certificate to be issued
